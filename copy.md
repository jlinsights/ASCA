## Part B. 이상 고객 프로필 (ICP)

### 4. ICP 요약

| ICP       | 세그먼트           | 페르소나                   | 핵심 페인                                       | 서비스 매칭                       |
| --------- | ------------------ | -------------------------- | ----------------------------------------------- | --------------------------------- |
| **ICP-1** | 문화예술 단체/협회 | 사무국장, 40~60대          | 엑셀 회원관리, 수동 홍보, 신청/심사 비효율      | 마케팅 자동화 + 데이터 파이프라인 |
| **ICP-2** | 갤러리/전시공간    | 대표/큐레이터, 30~50대     | 방문객 분석 불가, VIP 관리 부재, 문의 과부하    | 마케팅 자동화 + AI 챗봇           |
| **ICP-3** | 서예/캘리 교육기관 | 학원장, 30~50대            | 수강생 모집 의존, 수작업 관리, 이탈률 관리 부재 | AI 챗봇 + 마케팅 자동화           |
| **ICP-4** | 지자체/공공기관    | 문화예술과 주무관, 30~50대 | 성과 데이터 부재, 정량 입증 압박, 제한적 홍보   | 데이터 파이프라인 + AI 챗봇       |

### 5. ICP-1: 문화예술 단체/협회

**조직**: 사단법인, 비영리 문화단체 | 회원 100~5,000명 | 예산 연
1,000만~5,000만원 **디지털 성숙도**: 낮음 (엑셀 + 카카오톡 + 네이버 카페)

| 문제 (Pain)                                  | 심각도 | 현재 해결책               |
| -------------------------------------------- | :----: | ------------------------- |
| 회원 관리가 엑셀/수기 — 누락과 중복 빈번     |  HIGH  | 엑셀 + 카카오톡 단체방    |
| 전시/행사 홍보가 수동 포스팅에 의존          |  HIGH  | 네이버 카페, 블로그, 문자 |
| 신규 유입이 지인 소개에 편중 — 고령화        |  MED   | 오프라인 행사, 지인 추천  |
| 전시 참가 신청/심사/발표가 전화+이메일       |  HIGH  | 전화, 이메일, FAX         |
| 작가 활동 이력 데이터가 체계적으로 축적 안됨 |  MED   | 종이 서류, 개인 폴더      |

**소구 포인트**: "엑셀에서 벗어나 회원 관리를 자동화하세요. 전시 신청부터 결과
발표까지 원클릭."

### 6. ICP-2: 갤러리/전시공간 운영자

**조직**: 상업 갤러리, 대안공간, 아트센터 | 직원 2~15명 | 예산 연
500만~3,000만원 **디지털 성숙도**: 중간 (인스타그램 활용, 기본 웹사이트)

| 문제 (Pain)                                          | 심각도 | 현재 해결책             |
| ---------------------------------------------------- | :----: | ----------------------- |
| 방문객 데이터 수집/분석 불가 — 마케팅 효과 측정 못함 |  HIGH  | 방명록, 인스타 인사이트 |
| 작가 포트폴리오/작품 DB 미관리                       |  MED   | 구글 드라이브, 이메일   |
| VIP 컬렉터 맞춤 안내 어려움                          |  HIGH  | 수동 문자/이메일        |
| SNS 콘텐츠 제작에 과도한 시간                        |  MED   | 직원 수동 포스팅        |
| 전시 문의 응대에 인력 묶임                           |  MED   | 전화 + 카카오톡 수동    |

**소구 포인트**: "방문객 데이터를 자동으로 수집하고, VIP에게 맞춤 전시 초대를
보내세요."

### 7. ICP-3: 서예/캘리그라피 교육기관

**조직**: 서예학원, 캘리그라피 클래스, 문화센터 | 수강생 50~500명 | 예산 연
300만~2,000만원 **디지털 성숙도**: 낮음~중간

| 문제 (Pain)                               | 심각도 | 현재 해결책                      |
| ----------------------------------------- | :----: | -------------------------------- |
| 수강생 모집이 네이버/인스타 광고에만 의존 |  HIGH  | 네이버 플레이스, 인스타 광고     |
| 수강 신청/결제/출석 관리 수작업           |  HIGH  | 전화 예약, 계좌이체, 수기 출석부 |
| 수강생 이탈 징후 미파악 — 재등록률 하락   |  MED   | 감에 의존                        |
| 반복 문의(시간, 수강료, 주차) 시간 낭비   |  MED   | 전화, 카카오톡 수동              |

**소구 포인트**: "반복 문의는 AI가, 수강생 관리는 자동으로. 강사는 교육에만
집중하세요."

### 8. ICP-4: 지자체/공공기관 문화사업 담당

**조직**: 시/구/군청, 문화재단, 문화원 | 담당 1~5명 | 예산 사업별 500만~2억원
**디지털 성숙도**: 낮음 (공공기관 보수적)

| 문제 (Pain)                                     | 심각도 | 현재 해결책                 |
| ----------------------------------------------- | :----: | --------------------------- |
| 행사 참가자 모집/관리 수동                      |  HIGH  | 공문, 전화, 엑셀            |
| 성과 데이터(참가자, 만족도, 재방문) 수집 어려움 |  HIGH  | 종이 설문, 수기 집계        |
| 예산 대비 효과 정량 입증 압박                   |  HIGH  | 사진 + 보도자료 보고        |
| 문화 프로그램 홍보 채널 제한적                  |  MED   | 구청 홈페이지, 현수막, 전단 |

**소구 포인트**: "데이터로 증명하는 문화사업. 참가자 수집부터 성과 보고서까지
자동화."

## Part C. 엔터프라이즈 기능 카피

### 9. 관리 시스템 및 보안

**대상**: 내부 운영팀, 시스템 관리자

**시스템 터미놀로지**:

- **감사 로그(Audit Log)**: 시스템 내 모든 데이터 변경 및 접근 이력을 기록한
  무결성 기록
- **리스크 점수(Risk Score)**: 작업의 위험도를 0~1 사이로 수치화한 지표
- **에이전트 모니터(Agent Monitor)**: AI 서비스 에이전트들의 실시간 상태 및
  처리량 감시
- **보안 컨텍스트(Security Context)**: 사용자의 권한과 작업 환경을 결합한 검증
  상태

**알림 및 경고 카피**:

- `ACCESS_DENIED`: "접근 권한이 없습니다. 필요한 권한: {permission}"
- `RISK_ALERT`: "고위험 작업이 감지되었습니다. 계속하시려면 보안 인증이
  필요합니다."
- `AUDIT_TRACKED`: "이 작업은 보안 정책에 따라 감사 로그에 기록됩니다."
- `SYSTEM_HEALTH_LOW`: "시스템 자원 사용량이 임계치에 도달했습니다. 관제팀에
  알림이 전송되었습니다."

### 10. 서비스별 카피 (업데이트)

**대상**: ICP-1(Primary), ICP-2(Primary), ICP-3, ICP-4

**헤드라인 후보**:

- "전시 오픈부터 후기 수집까지, 하나의 흐름으로"
- "회원에게 딱 맞는 메시지를, 딱 맞는 타이밍에"
- "수동 포스팅은 끝. 전시 홍보를 자동으로"

**활용 사례**:

- 전시 오픈 자동 이메일/카카오 알림
- 회원 등급별 맞춤 콘텐츠 발송
- 수강 만료 전 재등록 유도 자동 메시지
- VIP 컬렉터 전시 초대 자동화
- 행사 후 만족도 조사 자동 발송

### 10. 데이터 파이프라인

**대상**: ICP-1(Primary), ICP-4(Primary), ICP-2

**헤드라인 후보**:

- "엑셀 30개를 하나의 대시보드로"
- "작가의 이력이 쌓이면, 협회의 자산이 됩니다"
- "데이터로 증명하는 문화사업 성과"

**활용 사례**:

- 엑셀 회원 데이터 -> 통합 DB 마이그레이션
- 작가별 활동 이력/수상 경력 자동 축적
- 전시 방문객 데이터 수집 및 트렌드 분석
- 문화사업 성과 지표 자동 대시보드
- 회원 이탈 예측 데이터 분석

### 11. AI 챗봇

**대상**: ICP-2(Primary), ICP-3(Primary), ICP-1, ICP-4

**헤드라인 후보**:

- "전시 문의의 80%는 AI가 답합니다"
- "24시간 응대, 0명의 추가 인력"
- "수강 문의부터 신청까지, 챗봇이 안내합니다"

**활용 사례**:

- 전시 일정/위치/입장료 문의 자동 응답
- 수강 신청/수업 시간/수강료 안내
- 회원 가입 절차 안내 및 서류 접수
- 작가 공모전 접수 안내
- 문화행사 참가 신청 접수 자동화
