## 0. 브리프 사용법

이 문서는 디자이너 1인이 단독으로 도록·포스터 작업을 시작할 수 있도록 압축한 1차
브리프입니다. 인쇄 사양·해상도·바인딩 같은 제작 결정은 디자이너의 권한이며, 이
문서는 **카피·시각 방향·금기**만 규정합니다.

작업 순서 권고:

1. `brand-guidelines.md §0~§3·§7~§8` 정독
2. `product-marketing-context.md` 톤·금기 어휘 확인
3. 본 브리프로 도록 또는 포스터 1종 시안 → 협회 사무국 1차 검토

## 1. 도록 디자이너 브리프

### 1.1 용도

협회 정기 전시·공모전·기획 전시의 공식 도록. 전시장 비치 + 작가·심사위원·후원자
증정 + 협회 영구 자산.

### 1.2 카피 매핑 (필수)

| 위치              | 카피 단계                                                                                      | 출처                           |
| ----------------- | ---------------------------------------------------------------------------------------------- | ------------------------------ |
| 표지              | 전시명 + 부제 + 한자 표지 `法古創新 · 人書俱老`                                                | `brand-guidelines.md §3.1`     |
| 권두언 (Foreword) | 4언어 병기 — 한국어 L3 + 中文 L3 + 日本語 L3 + English L3                                      | `brand-guidelines.md §1`       |
| 협회 소개         | 한국어 L3 (단독)                                                                               | `brand-guidelines.md §1.1`     |
| 작품 페이지 캡션  | 작가명·작품명·재질·크기·연도 + 작가 짧은 한 줄 (한국어 + 영어 병기 권장)                       | 사무국 제공                    |
| 후원·협력         | 협력 단체 로고·이름 (예: 삼성금융네트웍스, 예술의전당, 대한검정회, 서울특별시, 문화체육관광부) | `product-marketing-context.md` |
| 판권              | 협회 정보 + 사무국 연락처 + 발행일                                                             | 사무국 제공                    |

### 1.3 시각 방향

**색상 (brand-guidelines §7.1)**

- 본문: 백색·한지색 배경 + 먹색 본문 텍스트
- 강조: 인주색 (제목 1자 또는 도장 모티프 한정)
- 표지: 한지색 + 먹색 한자 표지 + 인주색 점 1개
- 금색은 의례성 강조에 한해 절제 사용 (표지 협회 인장 또는 권두언 머리 장식
  한정)

**서체 (brand-guidelines §7.2)**

- 한국어 본문: 명조 계열
- 중국어 본문: 송체·명조
- 일본어 본문: 明朝体
- 영어 본문: Serif
- 한자 표지(法古創新·人書俱老): 명조 또는 정자 해서풍 — 절제·균형
- 캘리그래피 흉내 디지털 폰트 회피

**여백·레이아웃**

- 작품 페이지는 작품 1점 단독 노출 — 여백을 충분히 살린다
- 작품 사진은 정면·자연광·여백이 보이도록 (작가 제출본 그대로 또는 사무국
  재촬영본)
- 그리드 위계: 표지 → 권두언 → 협회 소개 → 작품 → 작가 약력 → 후원 → 판권

### 1.4 이미지 가이드

- 작품 클로즈업 이미지를 표지·섹션 시작면에 배치 가능 (단, 작품의 일부 — 한 획
  또는 한 글자만)
- 한지 질감·먹 번짐을 배경 텍스처로 절제 사용
- 작가 작업 사진(손·붓·먹·종이) 섹션 사이 인덱스 페이지에 배치 가능

### 1.5 금기 (강한 제약)

- ❌ 화려한 배경·그라데이션·발광 효과
- ❌ 붓글씨 흉내 디지털 그래픽 (실제 작품 아니면서 서예처럼 보이는 디자인 요소)
- ❌ 어둡고 낡은 톤으로 "전통=낡음" 인상
- ❌ NFT·메타버스·암호화폐 결합 모티프 일체
- ❌ 영문 폰트로 동양풍 흉내 (예: brushy 동양 분위기 폰트)
- ❌ 마케팅적 자기 자랑 카피 ("최고의 서예 단체" 등) —
  `brand-guidelines.md §5.1` 참조

### 1.6 인쇄 사양 권고

- 디자이너 결정 영역. 다만 다음을 권고:
  - 무광 코팅 또는 한지 질감 표지 용지
  - 본문 용지는 모조지 또는 한지 풍 용지 (사진 인쇄 가능 등급)
  - 바인딩은 양장 또는 누드 제본 — 정기·기획 도록의 격조에 맞춰

## 2. 전시 포스터 디자이너 브리프

### 2.1 용도

협회 정기·기획 전시 공식 포스터. 전시장 외부·미술관 옥외·SNS·웹·초대장 축약형의
모자(母) 디자인.

### 2.2 전시 5종 카피 매핑

`brand-guidelines.md §3.1` 의 5종 가운데 1종 선택 또는 새 제목 생성. 어느 경우든
핵심 철학(法古創新 또는 人書俱老)이 부제·캡션 중 1곳에 반드시 포함.

| 옵션 | 제목                | 부제                                                       | 메인 카피                                                                 |
| ---- | ------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------- |
| 1    | 書境 — 새로운 지평  | 법고창신의 붓끝에서, 오늘의 서예를 다시 쓰다               | 한 획에 전통을 새기고, 여백에 오늘의 정신을 담습니다.                     |
| 2    | 法古創新 · 人書俱老 | 옛 법을 익혀 새로움을 열고, 글씨와 사람이 함께 깊어지는 길 | 글씨는 손끝에서 시작되지만, 그 깊이는 사람의 시간에서 완성됩니다.         |
| 3    | 문자, 정신이 되다   | 한글의 리듬, 한문의 깊이, 서예의 오늘                      | 문자는 기록을 넘어 정신이 되고, 서예는 시대를 잇는 예술이 됩니다.         |
| 4    | 線墨氣韻            | 선과 먹, 여백과 정신이 만나는 예술                         | 보이는 것은 선과 먹이지만, 남는 것은 기운과 정신입니다.                   |
| 5    | 書藝現代            | 전통을 쓰고, 오늘을 열다                                   | 오래된 문자는 낡은 것이 아니라, 시대마다 새롭게 읽히는 정신의 그릇입니다. |

### 2.3 4개국어 핵심 카피 (포스터 변형 또는 도록 표지)

| 언어    | 핵심 카피                                                  |
| ------- | ---------------------------------------------------------- |
| 한국어  | 한 획에 전통을 새기고, 여백에 오늘의 정신을 담습니다.      |
| 中文    | 一筆承古，空白生新。                                       |
| 日本語  | 一筆に伝統を宿し、余白に今を映す。                         |
| English | In One Stroke, Tradition; In Silence, the Spirit of Today. |

국제 교류 전시의 경우 4언어 모두 한 포스터에 병기. 국내 정기 전시는 한국어 +
한자 표지 중심.

### 2.4 필수 구성 요소

| #   | 요소                                                                  | 위치 권고                             |
| --- | --------------------------------------------------------------------- | ------------------------------------- |
| 1   | 회차·전시명 (예: "제23회 대한민국 동양서예대전 / 書境 — 새로운 지평") | 상단 또는 중앙 상부                   |
| 2   | 한자 표지 `法古創新 · 人書俱老`                                       | 시각 중심부 또는 하단 모티프          |
| 3   | 메인 카피 (위 §2.2 표)                                                | 제목 아래 1~2줄                       |
| 4   | 일시·장소·관람료·주최·후원                                            | 하단 정보 영역                        |
| 5   | 협회 로고·인장 모티프                                                 | 우측 하단 또는 협회 정체 영역         |
| 6   | 작품 1점 일부 클로즈업 (선택)                                         | 시각 자산화 — 한 획 또는 한 글자 단위 |

### 2.5 시각 방향

- 색상: 백색·한지색 베이스 + 먹색 본문 + 인주색 1점 강조 —
  `brand-guidelines.md §7.1`
- 서체: 명조 계열 본문 + 한자 표지는 정자 해서풍
- 여백 비율: 정보 영역 25% 이내, 시각·여백 75% — 서예의 호흡 원칙
- 작품 클로즈업 사용 시 작품 자체가 주인공, 그래픽 장식은 최소

### 2.6 금기 (강한 제약)

- ❌ 한자 표지를 회사 로고처럼 좌측 상단에 작게 처리
- ❌ 메인 카피에 자극적 어휘 ("혁신", "최고", "당신만의" 등)
- ❌ 작품 위에 텍스트를 겹쳐 작품 가독성 훼손
- ❌ 그라데이션 배경·네온·발광 효과
- ❌ 4언어 병기 시 한 언어가 시각적으로 압도하는 위계 (예: 영어만 크게)

### 2.7 SNS·웹 변형 가이드

- 정사각(1:1) 변형: 메인 카피 + 한자 표지 + 회차/전시명만 — 일시·장소는 캡션으로
  분리
- 세로(9:16) 변형: 한자 표지를 시각 중심 → 카피는 위·아래에 분산
- 영문권 SNS: §2.3 영문 카피로 교체

## 3. 디자이너 결정 영역

다음 항목은 디자이너의 권한이며 협회는 시안 검토 시 의견만 제시:

- 표지·포스터의 정확한 그리드·여백 수치
- 폰트 패밀리 최종 선택 (위 권고 범위 내)
- 인쇄 용지·코팅·바인딩
- 사진 보정 강도
- 도록 페이지네이션·인덱스 표기 방식
- 포스터의 일러스트레이션 사용 여부

---

## 4. 검토 절차

1. 1차 시안 (디자이너) → 협회 사무국 검토 (3 영업일)
2. 사무국 피드백 → 디자이너 2차 시안 (5 영업일)
3. 임원 회의 또는 큐레이션 위원 검토 → 최종 결정
4. 인쇄·게시 발주 (디자이너 또는 사무국)

각 단계에서 본 브리프와 `brand-guidelines.md` 가 평가 기준입니다. 시안이 이 두
문서와 어긋나면 그 이유를 디자이너가 설명하고 사무국이 결정합니다.

### 6.1 디자이너 1인 컨택용

```
제목: [동양서예협회] 도록·전시 포스터 디자인 의뢰 드립니다 — 1차 브리프 동봉

○○○ 선생님께,

평소 ○○ 작업에서 보여주신 ○○ 결을 인상 깊게 보고 있습니다.
선생님의 시선이 동양서예협회의 도록·전시 포스터 작업에 잘 맞을 것 같아
조심스럽게 의뢰의 말씀을 드립니다.

저희 협회는 法古創新·人書俱老의 정신으로
옛 법을 깊이 익혀 새로움을 열고,
글씨와 사람이 함께 깊어지는 서예 문화를 지향합니다.

첨부드린 1차 브리프(`dorok-poster-design-brief.md`)와
브랜드 가이드(`brand-guidelines.md`)를 먼저 살펴보아 주시고,
관심이 있으시면 1차 시안 일정·범위·견적을 회신 주시면 감사하겠습니다.

본 브리프는 카피·시각 방향·금기만 규정하고 있으며,
인쇄 사양·여백 수치·폰트 패밀리 최종 선택 등 제작 결정은
선생님의 권한임을 미리 말씀드립니다.

검토 절차는 1차 시안 → 사무국 검토 → 2차 시안 → 임원 회의로 진행됩니다.

회신 기다리겠습니다.

— 동양서예협회 사무국 ○○○ 드림
   info@orientalcalligraphy.org
```
